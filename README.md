# 👑 Customer Lifetime Value (CLV) Prediction System

A Machine Learning web application that predicts Customer Lifetime Value using RFM Analysis on the UK Online Retail dataset.

## 🎯 Problem Statement
Businesses struggle to identify which customers are most valuable. This system solves that by predicting CLV and segmenting customers into actionable groups.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit

## 📊 Features
- RFM Analysis (Recency, Frequency, Monetary)
- Customer Segmentation (Champion, Loyal, At Risk, Lost)
- CLV Prediction using Random Forest Regressor (R² = 0.966)
- Customer Classification — High / Medium / Low Value (Accuracy = 98.8%)
- Cohort Analysis — Retention Heatmap
- Interactive Business Recommendations

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor
- Logistic Regression
- Random Forest Classifier

## 📁 Dataset
Online Retail II UCI Dataset — UK based online store transactions (2009-2011)

## 🚀 How to Run
```bash
streamlit run customer.py
```
